## 💡 프로젝트 소개
Next.js를 학습하기 위한 프로젝트입니다.
Google Calender를 모티브로 만든 ToDoList입니다.

<br>
## 📋 주요 기능 목록

<br>

###  🚩 메인 화면
로그인
```
- 로그인 기능 (Passport Session)
- 로그아웃
``` 

###  🚩 회원가입, 아이디찾기
회원가입, 아이디찾기
```
- 회원가입
- 아이디찾기
 ``` 

###  🚩 비밀번호 찾기
비밀번호 찾기
```
- 비밀번호 찾기
- 이메일 인증(Gmail API)
- 임시 비밀번호
 ``` 

###  🚩 ToDoList
일정표
```
- 달력
- 메모장
- 일정 생성하기 (By Click For 사용자 친화적)
- 일정 옮기기 (By Drag For 사용자 친화적)
- 일정 사이즈 변경 (By Drag For 사용자 친화적)
- 타임바 (08:00 ~ 24:00) 
 ``` 

###  🚩 일정 세부사항
일정 세부사항
```
- 일정 수정 (TextEditor By React Quill)
- 일정 삭제
``` 

###  🚩 알람
알람
```
- 자명종 소리
- 일정 알람
- 시간
```
